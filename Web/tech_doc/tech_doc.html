What is the Arduino?

The Arduino commonly refers to the product "Arduino Uno". But is collectively a platform for open source electronic modifications with a suite of microcontroller or microprocessor boards capable of completing many tasks required by the user. The Arduino is also a C/C++ library to interface with boards (library refers to external code that can be used) to program the Arduino board. This means programmers that use Arduino will also need to understand some basic C/C++ syntax.

The Arduino Uno

The guide will use the Arduino IDE (Independent Developing Environemnt) to program Arduino boards and can be downloaded on the Arduino website. In short, the Arduino is a:

  • Hub for all people to learn electronics in a open source environment
  • Collection of electronic boards
  • Software IDE to develop code for the boards

Variables and data types

Here is a broad overview of some of the data types that exist in the C language (not a strict subset of C++) and we will cover most of the relevant ones below.

The C/C++ language has five primitive (meaning basic) data types used to store information, and they are:

  • char //Used to store alphanumeric characters from ASCII encoding
  • int //Used to store only integers with a finite range 
  • long //Used to store only integers with a greater finite range than ints
  • float //Used to store decimal numbers, this can also include integers
  • void //A data type that is empty, useful with derived data types

The double back slashes "//" are known as comments. These are notes for others to read and are ignored by the program.

Storing numbers:

The int, long and float are used to store numbers. But the method by which they are stored in the computer are different. Specifically, int and long can only store integers by representing decimal numbers as a bit sequence. See the diagram below and link to understand the calculations. Note that the range of integer depends on the processor used. The Arduino Uno with the ATMega328 chip stores an int as a length of 16 bits and a long as 32 bits.

Note that you raise two to the power of the position from the right, starting from zero and multiply it by 1 or 0 depending on the bit. Image was courteously shared by JavatPoint

The float stores decimal numbers. While you can store an integer such as: 9, it is stored differently on the computer than an int or long. See the diagram of a float data type below to understand it's inner workings.

A floating point data type consists of:

  • A sign: that tells if the number is positive or negative
  • An exponent: the number that raises the power of 10
  • A fraction: a x = 1 2 n .

The int or long does not have three parts to store numbers like the float and explaining calculations for the float is not needed to understand the Arduino completely. But more information can be found here.
